### **Project Overview**
A modern, responsive **Product Review System** that allows users to browse products, submit reviews with star ratings, and ensures **quality control** through automated and manual moderation of user inputs.

This project demonstrates **secure handling of user-generated content** — a critical aspect of modern web applications.

---

### **Key Features**

#### **User Side**
- Browse multiple products with images
- Submit reviews with 1–5 star rating
- Real-time form validation
- Responsive design (mobile-friendly)

#### **User Input Moderation** (Main Focus)
- **Automated Moderation:**
  - Profanity / Abusive word detection
  - Review length validation (10–500 characters)
  - Spam detection (repeating characters)
  - Empty or invalid input check
- **Status Management:** Pending / Approved / Rejected
- **Manual Moderation** by Admin

---

### **Tech Stack**
- **Frontend:** HTML5, CSS3, Tailwind CSS, JavaScript
- **Backend:** PHP 8+
- **Database:** MySQL
- **Architecture:** MVC-style with clean API structure

---

### **How to Run Locally**

1. Clone or download the repository
2. Copy folder to `htdocs/product-review-portal` (XAMPP)
3. Start Apache and MySQL
4. Import the provided SQL file in phpMyAdmin
5. Update database credentials in `config/db.php`
6. Open browser and go to:  
   `http://localhost/product-review-portal`
